# 🍳 Cocinando Rico Bot

Bot automatizado de publicación de recetas culinarias para Facebook. Publica recetas deliciosas de todo el mundo con imágenes atractivas y formateo profesional.

## ✨ Características

- 🌍 **Recetas internacionales** de TheMealDB (gratuito)
- 🗣️ **Soporte en español** con Edamam API (opcional)
- 🖼️ **Imágenes automáticas** (descarga o generación personalizada)
- 🏷️ **Hashtags inteligentes** según categoría y origen
- 📊 **Anti-duplicados** con sistema de historial robusto
- ⏰ **Control de frecuencia** de publicaciones
